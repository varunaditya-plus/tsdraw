## Feature highlights

tsdraw is an infinite canvas drawing package with modern design at its heart.

- **drawing**: tsdraw supports styluses, and comes with an easily customizable ui
- **tremendously small**: tsdraw is made to be small, and load into your page instantly when you need it.

## Principles

these are some core principles I will follow while building tsdraw

- **free, forever**: unlike other drawing packages similar to tsdraw, which have commercialised over time, I want to keep tsdraw free and 100% open source under the `GPLv3` license, meaning anyone can make their own tsdraw.
- **design**: for this project, I want to focus a lot on developing my UI/UX skills to make this the most enjoyable online drawing app. period.
- **modularity**: i want to make this package super modular, so you will be able to build it for you.
